Build an AI-powered recruiting submission workspace for staffing agencies.

Recruiters should be able to enter a candidate once, upload or paste their resume, add notes, and prepare submissions for multiple client portals. The MVP should include a dashboard of candidates, a candidate detail page, a job/client submission checklist, AI-generated submission summaries, duplicate submission warnings, status tracking across clients, and an admin/settings area for managing client portal requirements.

The app should support multiple recruiters on the same team, basic login, persistent candidate records, a backend API, a database, and a clean modern frontend. It should also include audit/history notes so recruiters can see what was changed and when.

For the first demo slice, prioritize something visible and useful, not the entire product.
